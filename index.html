How to get your API Keys

To use this monitor, you need two free keys. These keys act like passwords that allow this site to talk to legislative databases and AI models on your behalf.

1. LegiScan API Key

LegiScan provides the raw legislative data (bill text, dates, and status).

  1. Go to LegiScan.com and register for a Free Personal Account.
  2. Once logged in, go to the API Summary or Settings page.
  3. Copy your API Key string.

2. Google AI Studio (Gemini) Key

This key powers the Gemini 3 Flash model that summarizes the bills.

  1. Visit Google AI Studio.
  2. Sign in with your Google account.
  3. Click on "Get API key" in the sidebar.
  4. Click "Create API key in new project".
  5. Copy the key. Note: In 2026, ensure you are using a key that supports v1beta models for Gemini 3.
Security Note: These keys are stored only in your browser's memory while the page is open. They are not saved to any server. If you refresh the page, you will need to paste them again.
